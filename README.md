# 🎵 Music Player

This is a simple music player I built using HTML, CSS, and JavaScript. The idea was to create something that looks clean and modern (kind of inspired by Spotify) while still keeping the code easy to understand.

---

## What it does

* Play and pause songs
* Skip forward and backward
* Show song progress and let you seek through it
* Display song name and artist
* Clean UI with album-style design

## How I built it

I used:

* HTML for the structure
* CSS for styling (layout, colors, buttons, etc.)
* JavaScript to control the audio and handle interactions

Instead of using the default audio player, I removed the built-in controls and created my own UI.

---

## How to run it

1. Download or clone the project
2. Open it in VS Code
3. Run `index.html` using Live Server

That’s it — the player should open in your browser.

---

## Notes

* The audio is controlled using JavaScript (`play()`, `pause()`, etc.)
* The progress bar is synced with the song time
* You can customize songs and images easily by changing the files
